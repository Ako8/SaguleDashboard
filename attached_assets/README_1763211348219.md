## ğŸ†˜ Troubleshooting Common Issues

**Issue:** "401 Unauthorized"
**Solution:** Check if JWT token is properly stored and sent in headers

**Issue:** "CORS Error"
**Solution:** Ensure backend has proper CORS configuration

**Issue:** "File upload fails"
**Solution:** Check Content-Type is multipart/form-data and entityId/entityType are in query params

**Issue:** "Calendar sync not working"
**Solution:** Verify iCal URL is valid and accessible, check CalendarSource enum value

**Issue:** "Payment creation fails"
**Solution:** Ensure all required query parameters are sent (amount, currency, returnUrl)
